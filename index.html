Happy Taps

Happy Taps is a project that I created to gain familiarity with different technologies. It provides a backend service for a Slack app that generates happy hour suggestions in a particular area based on Yelp data.

Happy Taps is deployed as two containerized apps running on GCP's Cloud Run platform. It leverages Pub/Sub to decouple front end requests from data processing for scalability. Additionally, Cloud Datastore is used as a caching layer for improved responsiveness.
